# One Page Renewal
개인프로젝트

## 🖥️ 프로젝트 소개
원페이지 유형의 웹사이트를 레퍼런스로 참고하여 리뉴얼해서 디자인한 반응형 사이트입니다.
<br>

## 📌 주요 기능
#### 자바스크립트 플러그인으로 다양한 시각적 디자인 요소 추가
- morphsvg.js (SVG 애니메이션 효과)
- jquery.ripples.js (마우스 위치 좌표에 따른 애니메이션 효과)
- smoothScroll.js (스크롤링 동작을 부드럽게 만들어주는 효과)
